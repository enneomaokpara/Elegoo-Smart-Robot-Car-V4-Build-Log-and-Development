# Elegoo-Smart-Robot-Car-V4-Build-Log-and-Development
Arduino-based autonomous robot car using sensors, motor control, and embedded systems.

## Overview
This project documents the assembly and development of an Elegoo Smart Robot Car V4.

## Hardware
- Arduino Uno
- Smart Car Shield V1.1
- 4 DC Gear Motors
- Battery Pack
- HC-SR04 Ultrasonic Sensor
- Raspberry Pi 5 (future integration)

## Progress
- [x] Assembled chassis
- [x] Mounted motors
- [x] Installed wheels
- [x] Connected Smart Car Shield
- [x] Connected battery system
- [x] Verified Arduino communication
- [ ] Motor testing
- [ ] Obstacle avoidance
- [ ] Camera integration
